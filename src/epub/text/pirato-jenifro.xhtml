Pirato Ĵenifro

Sinjoroj, hodiaŭ vi vidas min glaslavi
Kaj mi lit-faras por gastaro
Kaj vi donos al mi la pencon kaj mi rapide dankos
Kaj vi vidos ĉifonojn kaj hotelaĉon
Kaj ne konas, al kiu vi diras
Kaj ne konas, al kiu vi diras
Sed iun vesperon venos ŝrikoj el haveno
Kaj oni demandos: Kiaj ŝrikoj?
Kaj oni vidos min rideti ĉe glasoj
Kaj diros: “Pri kio ridetas?”

Kaj ŝipo kun ok veloj
Kaj kun kvindek kanonoj
Kuŝos ĉe kajo

Oni diras: Glasviŝu, fraŭlino
Kaj oni ĵetas mi pencon
Kaj penco estas prenita, kaj lito farita!
(Ja neniu plu dormos en ĝi, ĉi-nokte.)
Kaj eĉ nun ne scias, kiu mi estas.
Kaj eĉ nun ne scias, kiu mi estas.
Sed iu vespero estos havene bruego
Kaj oni demandas: Kia bruego?
Kaj oni vidos min stari post fenestro
Kaj diras: Pro kio malice?

Kaj ŝipo kun ok veloj
Kaj kun kvindek kanonoj
Bombardos la urbon

Sinjoroj, ilia rido nepre tuj haltos
Ĉar la muroj disfalos
Kaj la urbo detrue ebeniĝos
Nur hotelaĉon, atako preterlasos
Kaj oni diras: Apartulo tie?
Kaj oni diras: Apartulo tie?
Kaj tiu nokte estos ŝrikoj ĉirkaŭ hotelo
Kaj oni diras: Kial ci-hotelon?
Kaj oni vidos min matene transpaŝi pordon
Kaj diros: Tiu loĝis tie?

Kaj ŝipo kun ok veloj
Kaj kun kvindek kanonoj
Ekflagos la maston

Kaj cento venos marborden ĉirkaŭ tagmezo.
Kaj trete paŝos en la ombroj.
Kaj ekkaptas ĉiun lastan el ĉiu pordo
Kaj katenas lin kaj kondukas lin al mi:
Kaj demandas: Kiun ni mortigu?
Kaj min demandas: Kiun ni mortigu?
Kaj tiu tagmeze trankviliĝos havene
Se demandus, kiu mortu
Kaj tiam vi aŭdos, ke mi diros: Ĉiuj!
Kaj dum kapo falas, mi diras: Ha!

Kaj ŝipo kun ok veloj
Kaj kun kvindek kanonoj
Kun mi malaperos
